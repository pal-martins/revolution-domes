# revolution-domes
Código en javascript para el cálculo de la geometría de domos de revolución.
Se pueden calcular las geometrías de ZOMES, L-DOMES (LOW DOMES) Y TIPIS (CONOS)
A parte de dibujarnos una vista en planta y alzado de la geometría,
también nos genera los valores de la geometría, entre ellos la longitud de las barras horizontales y diagonales de cada nivel. 
Con estos valores se puede construir perfectamente esta geometria tridimensional con barras de cualquier material
